<h1>Pokemon Card Project</h1>

Pokemon card, pokemon kart bilgilerini içeren ve isimleriyle arama yapmanızı sağlayan basit bir web uygulamasıdır.

<h2>Kullanılan Teknolojiler</h2>

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

Responsive yapıda tasarlandı.
